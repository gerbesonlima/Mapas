Manual de Uso do Site - Mapas dos Territórios

1. Acessando os Mapas

Assim que você entrar no site, verá uma lista organizados por grupos a ordem esta seguindo a ordem dos territórios e não dos grupos.

2. Abrindo os Territórios

Clique no título do grupo para expandir a lista de territórios. Depois, clique no nome de um território para abrir o mapa correspondente no Google Maps. Isso permitirá que você veja a área delimitada daquele território.

3. Fechando a Lista de Territórios

Para esconder a lista de territórios de um grupo, basta clicar novamente no título do grupo.

Dicas Adicionais

É necessário ter o aplicativo Google Maps instalado para visualizar os territórios.

Ao abrir um mapa pela primeira vez, pode ser que a demarcação do território não apareça de imediato. Caso isso ocorra, basta clicar em "Tentar novamente".

Certifique-se de estar conectado à internet para acessar os mapas.

Agora você está pronto para usar o site e encontrar os territórios com facilidade!
